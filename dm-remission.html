ต้องสร้าง Firebase Project ก่อน (console.firebase.google.com)
Firebase Config (JSON)
วิธีตั้งค่า (ทำครั้งเดียว):
1. ไป Firebase Console → สร้าง Project ใหม่
2. เพิ่ม Web App → Copy firebaseConfig
3. ไป Firestore Database → สร้าง Database (Test mode)
4. วาง Config ด้านบน แล้วกด "เชื่อมต่อ"
5. ทำซ้ำขั้นตอนที่ 4 ทุกเครื่องในคลินิก
